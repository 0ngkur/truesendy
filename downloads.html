Frequently Asked Questions
Do I need an API key to use the app? ▼
Yes. The desktop app requires an Agency Plan API key to verify emails. You can purchase the Agency Plan from our pricing page and generate your key from the dashboard.
How many emails can I verify? ▼
Each API key comes with 100,000 verification credits. Each email verified uses 1 credit. When you run out, purchase a new key from your dashboard.
What file formats are supported? ▼
The app supports CSV, TXT, Excel (.xlsx, .xls), PDF, and Word (.docx) files. Emails are automatically extracted from any text content.
Does it work offline? ▼
No. The app connects to truesendy.com to validate your API key and track credits. The actual email verification happens on our servers.
What operating systems are supported? ▼
Windows 10 and Windows 11 (64-bit). The installer creates a desktop shortcut automatically.
